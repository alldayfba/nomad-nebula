Setup

4 Steps to Get Started

1

Get Claude Pro or Max

Go to claude.ai → Subscribe ($20/mo Pro or $100/mo Max)

2

Install Claude Code

Open Terminal → npm install -g @anthropic-ai/claude-code

3

Launch It

Type "claude" in terminal → Authenticate with your account

4

Start Talking

No setup files. No config. Just ask it things in plain English.

It's this simple

# Step 1: Install (one time)
$ npm install -g @anthropic-ai/claude-code

# Step 2: Launch
$ claude

# Step 3: Ask it anything
> Hey Claude, I sell on Amazon FBA. My niche is baby products.
> My buy criteria: 40%+ ROI, under $20 buy price, BSR under 50K.
> Find me 10 products to source this week.

# Claude remembers this forever. Next time just say "find products"
