## Targeting Notes

### Phase 1: Testing ($75-$150/day)

**Interest Targeting (3 ad sets):**
- **Ad Set A:** Amazon FBA, Fulfillment by Amazon, Amazon Seller, Amazon Seller Central
- **Ad Set B:** Side hustle, Passive income, Online business, E-commerce, Dropshipping
- **Ad Set C:** Entrepreneurship, Small business, Financial freedom, Work from home

**Demographics:**
- Age: 22-45
- Gender: All
- Locations: US, UK, Canada
- Language: English

**Exclusions:**
- Current students (upload customer list)
- Past purchasers (upload buyer list)
- Website visitors who already registered (pixel event)

### Phase 2: Scaling

**Lookalike Audiences:**
- 1% LAL of existing student list (highest value)
- 1% LAL of webinar registrants
- 1% LAL of website visitors (90 days)
- 2-3% LAL of video viewers (50%+) for broader reach

**Broad Targeting:**
- Once a winning ad is identified, test with no interests (age 22-45, US only)
- Let Meta's algorithm optimize on registration event

### Phase 3: Retargeting

**Warm Audiences:**
- Landing page visitors (did not register) — last 7 days
- Video viewers 50%+ — last 14 days
- Facebook/IG engagers — last 30 days
- Registered but did not attend — next webinar invite

**Retargeting ad copy:** Shorter, more direct. Reference that they already saw the content. "Still thinking about it?" angle.

## Creative Testing Framework

### Testing Protocol

1. **Launch:** Run all 3 angles simultaneously at $25-50/day each (1 angle per ad set)
2. **Each angle gets 2-3 creative variations:** Different hooks, same body. Test thumbnail vs no thumbnail on Reels.
3. **Minimum data:** Let each ad spend at least $50-$75 before making decisions (need 500+ impressions minimum)
4. **Kill criteria:** Pause at 3 days if CPA (cost per registration) exceeds 2x your target CPL
5. **Scale winners:** Increase budget by 20-30% every 3 days on ads beating target metrics
6. **Creative fatigue:** Rotate creative every 2-3 weeks or when CTR drops below 1%

### Target Metrics

| Metric | Target | Kill Threshold |
|---|---|---|
| CPL (registration) | $5-$15 | >$25 |
| CPA (booked call) | $75-$200 | >$350 |
| CTR (link click) | >1.5% | <0.8% |
| CPM | <$17 | >$30 |
| Hook rate (3-sec views) | >30% | <15% |
| Hold rate (ThruPlay) | >25% | <10% |
| Landing page CVR | >25% | <15% |

### Scaling Rules

- Never increase budget more than 30% in a single day
- If CPA spikes after scaling, hold budget steady for 48 hours before adjusting
- At $300+/day on a single ad, duplicate into new ad set rather than increasing budget (avoid re-entering learning phase)
- Maintain 3-5 active creatives at all times to prevent audience fatigue
- Refresh UGC footage monthly — new talking head, same proven scripts
