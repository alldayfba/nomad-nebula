## Core Philosophy

1. **Speed > perfection.** Don't spend 30 hours setting up OpenClaw when Perplexity or Co-Work does 90% of the job in 30 minutes.
2. **Markdown files are the new prompts.** You don't need to be a great prompter — you need great markdown SOPs.
3. **Automate the mundane. Do the work yourself.** Claude Code for actual client work. Co-Work/Dispatch for routine tasks.
4. **Pay-to-play.** AI is becoming pay-to-play. Pay for Claude Max and get everything. Don't cheap out on tools that multiply your time.
5. **Your job is to approve and monitor, not execute.** Build systems where you just click approve.

## Tool Routing Framework

```
Task is deep client work → Claude Code (terminal, multi-session)
Task is routine/automated → Co-Work + Dispatch OR Perplexity Computer
Task needs multi-model orchestration → OpenClaw
Decision on tool → whichever is easiest to set up for your skill level
```

**Dispatch mode:** Co-Work only (not Claude Code). Requires Mac Mini on / computer awake for local file access.

**Remote Control:** Claude Code feature — continue sessions from phone. Kabrin Ubers downtown specifically to work via Remote Control.

**Why terminal over IDE:**
- Multiple named sessions (organize by project/task)
- Side-by-side terminals for adjacent work items
- Stronger slash commands

## Automation Framework (Kabrin's Recommended Process)

1. Write down every task that takes up your time that doesn't need to
2. Open Claude Code, brain dump everything
3. Say: "I need you to build a plan for what I can automate through Co-Work or Perplexity"
4. Claude builds the plan → deploy via Co-Work/Dispatch
5. Your job = review quality, click approve, move forward

**Client comms example:**
- Perplexity/Co-Work checks all Slack channels every hour
- Drafts responses based on context
- Sends draft to Telegram → one-click approve or edit
- Sends automatically

## Ads & Funnel Principles

- **Congruency is everything:** Ad copy → landing page → VSL must all match. Mismatch kills conversions.
- **Creative fatigue:** After 60-90 days same creative = ad fatigue. Check age of creatives.
- **Funnel tracking:** Track each step: landing page hits → app starts → app completions → calls booked → shows
- **Attribution:** Hyros for paid. ClickFunnels native analytics solid. Webflow needs custom setup.
- **If lead flow dies:** First check ad spend (was it cut?), then check congruency, then check creative age
- **Heat maps:** Install on landing page to see where drop-off happens
