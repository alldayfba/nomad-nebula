## Available Tools

| Tool | Script | Purpose |
|---|---|---|
| Sourcing Pipeline | `execution/run_sourcing_pipeline.py` | End-to-end product sourcing |
| Retail Scraper | `execution/scrape_retail_products.py` | Scrape retail sites for product data |

## Access Restrictions

- **Read:** All directives, SOPs, memory files
- **Write:** `.tmp/` directory only (intermediate outputs)
- **Execute:** Only scripts listed above
- **No access to:** Client files, outreach bots, agency data
